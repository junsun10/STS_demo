### STS_demo

NAVER boostcamp lv1 STS 대회에서 snunlp/KR-ELECTRA-discriminator를 fine-tuning한 단일 모델 사용

문장1, 문장2를 입력받아 pearson correlation 측정


```

streamlit run app.py
```
